# Demo Sunburstchart

This demo application shows the capabilities of the TSunburstChart component.

![](preview.png)

General features:
 - Fully costumizable
 - based on TGraphicControl -> no other dependencies 
 - Properties: <br>
   * AngleOffset <br>
   * Color <br>
   * InitialArc <br>
   * LevelMargin <br>
   * PieRadius <br>
   * ShowHint
 - Eventhandlers: <br>
   * OnAfterSegmentPaint <br>
   * OnAfterPaint: <br>
   * OnBeforeSegmentPaint <br>
   * OnClick <br>
   * OnDeleteElementsUserData <br>
   * OnDblClick <br>
   * OnLoadUserData <br>
   * OnMouseUp <br>
   * OnMouseMove <br>
   * OnMouseDown <br>
   * OnSaveUserData <br>
   * OnShowHint <br>
   * OnResize
